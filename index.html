👋 Welcome to Recipe Finder

This is a simple, fast recipe discovery app built with HTML, CSS, and vanilla JavaScript. It uses the free TheMealDB API to fetch real recipes.

✨ Features

  • Search by name – type a dish (e.g., "pizza") and see matching meals.
  • Browse categories – click a category chip to see all meals in that category.
  • Detailed view – click any meal card to open a popup with ingredients, measurements, and cooking instructions.
  • Responsive design – works perfectly on mobile, tablet, and desktop.

🧑‍🍳 How to use

  1. On the Home page, either search for a meal using the input box or click one of the category buttons.
  2. A grid of meal cards will appear. Click on any card that looks tasty.
  3. A detailed popup shows the full recipe, including all ingredients and steps.
  4. Close the popup by clicking the ✕ button or clicking outside it.
  5. Switch back to the Info page anytime to read this guide again.

Enjoy cooking! 🍽️
